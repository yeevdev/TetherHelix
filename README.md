### How to execute?

For pycharm users, skip to number 2.

At the terminal(UNIX/LINUX supported)
1. enableing python virtual environment + requirements.txt installation
```
./tetherenv.sh -e
source .venv/bin/activate
```
2. launch
```
./tetherenv.sh run
# or just simply
python3 ./main.py
```
